# domchecker
Script de Python que sirve para comprobar los dominios que se encuentren operativos a partir de un archivo .txt con todos los dominios proporcionados

## INSTALACIÓN

### WINDOWS

```
pip install aiohttp
```

### LINUX

```
apt install python3-aiohttp
```
